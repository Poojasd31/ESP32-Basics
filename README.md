# ESP32-Basics
ESP32-Basics Information

## What is ESP32?
Ans: ESP32 is a low-cost, low-power, 32-bit microcontroller that integrates Wi-Fi and Bluetooth connectivity on a single chip.
It is widely used in embedded systems, Internet of Things (IoT), and smart device applications.

## Features of ESP32:
1. Built-in Wi-Fi (802.11 )
2. Bluetooth and Bluetooth Low Energy (BLE)
3. Multiple GPIO pins
4. ADC (12-bit), DAC, PWM
5. Communication interfaces: UART, SPI, I2C, CAN.
6. Supports FreeRTOS (real-time operating system)
7. Low-power modes for battery-operated devices

## Programming Support
1. Embedded C/C++
2. MicroPython
3. Lua

## Applications of ESP32
1. Smart home automation
2. IoT monitoring systems
3. Wearable devices
4. Wireless sensors
5. Industrial automation
6. Robotics
